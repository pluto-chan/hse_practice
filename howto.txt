Скрипт позволяет подобрать коэффициенты для функции известного вида по изображению графика.

Требования к изображению:
— График должен быть отцентрирован, т. е. начало координат находится в центре изображения
— На изображении должен присутствовать только график

Для запуска требуются:
— argparse
— matplotlib
— numpy
— scipy
— sympy

Параметры запуска
При запуске необходимо указать ряд обязательных параметров
1. imname — путь к изображению (с расширением)
2. xax — длина оси Х в координатной сетке
3. yax — длина оси Y в координатной сетке
4. func — вид функции

На данный момент поддерживаются следующие функции:
— lin — linear, k * x + b
— sqr — square, A * x^2 + B * x + C
— sqrt — square root, (k * x + b)^(1/2)
— hyp — hyperbole, A / (B * x + C) + D
